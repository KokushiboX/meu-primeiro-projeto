h1 até h6 são títulos, conteúdo vísivel tem que estar entre a tag body, p parágrafo, span pequenos trechos, e div especificam texto ou bloco, em e strong pra enfatizar o texto, br pra quebras de linha, ol para listas numeradas, ul para listas não ordenadas marcadas com marcadores, img pra imagem e video para video, links usados depois do src em ambos, video usa o widith e height como largura e altura, depois tem que ter o controls, alt depois da imagem para descrição, !DOCTYPE html como primeira linha do código sempre, html contendo todo o código html, title fica no head, a é usado para criar links para páginas interna ou externas, usando o href para o a tendo target _blank depois, e o id para qualquer outro, comentários são escristo com !-- comment --, table para tabela, tr para as linhas da tabela, td para células de dados, th para os cabeçalhos da tabela, scope= col é coluna, scope- row pra linha, o efeito de borda na tabela é feito com CSS (não desbloqueado ainda), colspan no td para fazer ele se estender por colunas, rowspan no td para fazer ele se estender por linhas, tbody para dividir a tabela em seções, thead para os cabeçalhos
